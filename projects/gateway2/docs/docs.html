    EnvoyBootstrap

    (Appears on:EnvoyContainer)

    Field Description
    logLevel
    string

    Envoy log level. Options include “trace”, “debug”, “info”, “warn”, “error”, “critical” and “off”. Defaults to “info”. See https://www.envoyproxy.io/docs/envoy/latest/start/quick-start/run-envoy#debugging-envoy for more information.
    componentLogLevels
    map[string]string

    Envoy log levels for specific components. The keys are component names and the values are one of “trace”, “debug”, “info”, “warn”, “error”, “critical”, or “off”, e.g.

    ```yaml
componentLogLevels:
upstream: debug
connection: trace
```

    These will be converted to the --component-log-level Envoy argument value. See https://www.envoyproxy.io/docs/envoy/latest/start/quick-start/run-envoy#debugging-envoy for more information.

    Note: the keys and values cannot be empty, but they are not otherwise validated.
